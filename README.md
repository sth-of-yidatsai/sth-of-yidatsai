# 蔡易達（YI-DA TSAI）

Taipei-based Visual Designer & Frontend Developer  
Typography · Editorial Design · Open Source

---

蔡易達（YI-DA TSAI）是一位來自台北的視覺設計師與前端工程師，  
專注於字體設計（Typography）與編輯設計（Editorial Design），  
致力於在視覺語言與數位體驗之間建立清晰且有結構的連結。

他的創作涵蓋字體設計、平面設計與網頁前端開發，  
並持續為開源字型與設計相關專案做出貢獻。  

在設計過程中，他重視版面結構、排版邏輯與視覺節奏，
並在內容與形式之間建立清晰的關聯。

目前，蔡易達持續探索設計與技術的交界，  
持續探索如何透過數位媒介延伸字體與編輯設計的可能性。

# YI-DA TSAI

YI-DA TSAI is a Taipei-based visual designer and frontend developer,  
specializing in typography and editorial design.

His work spans type design, graphic design, and web development,  
with ongoing contributions to open-source type and design-related projects.

His approach emphasizes layout systems, typographic logic, and visual rhythm.

Currently, he explores the intersection of design and technology,  
extending typography and editorial thinking into digital media.
